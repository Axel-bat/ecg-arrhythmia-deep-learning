# ECG Arrhythmia Classification with Deep Learning

## Objective
Build an end-to-end deep learning pipeline to classify cardiac arrhythmias from ECG signals.

## Dataset
- MIT-BIH Arrhythmia Dataset (PhysioNet)

## Methods
- Signal preprocessing (filtering, normalization)
- 1D Convolutional Neural Network
- Evaluation with F1-score and confusion matrix
